4. Makey Makey. ¿Cómo funciona?

Aprendiendo sobre Makey Makey

Makey Makey es una placa electrónica que permite conectar objetos cotidianos para controlar dispositivos electrónicos. Se puede usar para crear proyectos interactivos y divertidos. 

Características :

 
- Permite convertir objetos cotidianos en controladores
- Se puede usar para crear proyectos interactivos y divertidos
- Se conecta a un ordenador a través de un cable USB
- Se puede usar para enviar órdenes al ordenador
- Se puede combinar con internet o Scratch
- Se puede usar para crear paneles táctiles

A continuación se muestran algunas aplicaciones:

- Tocar un piano de plátano
- Jugar videojuegos con plastilina
- Crear una pista de baile interactiva con papel de aluminio
- Convertir casi cualquier superficie en un mando para juegos
- Convertir casi cualquier cosa en un teclado

Elementos Kit Makey Makey

Para usar Makey Makey necesitamos lo siguiente:

- 1 Placa Makey Makey

- Pinzas cocodrilo

- Cable USB

- Ordenador
